# Nepify

Modern e-commerce platform built with Go and React.

## Tech Stack

- **Backend**: Go, PostgreSQL
- **Frontend**: React, TypeScript, Tailwind CSS
- **Auth**: Clerk

## Getting Started

### Backend
```bash
cd apps/backend
make run
```

### Frontend
```bash
cd apps/frontend
npm install
npm run dev
```

### Docker
```bash
docker-compose up
```
